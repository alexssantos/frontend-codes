Pureza de funções

Para entendermos melhor o uso de componentes em angular precisamos entender a idéia de funções puras e impuras. Para isso vamos usar os scripts 1oPasso.js e 2oPasso.js e o botão calcular que vai nos retornar 10 números diferentes entre 40 e 100.




Veja agora o script do 1o passo, nele temos a função que é chamada pelo nosso botão executando todas as tarefas, essa é uma típica função impura.

Troque agora no arquivo index.html o valor do script src="1oPasso.js" para src="2oPasso.js. Abra o script 2o passo e analise o código."

No 2o passo a função pura criarVetorAleatório, administra sua atividade de criar um vetor de números aleatórios não repetidos independente de seu contexto. Não importa se os dados vêm de caixas de texto ou do console, a função mantém a sua independência.
